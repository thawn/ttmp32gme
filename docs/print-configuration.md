# Printing

## Print Process

1. Select albums in library
2. Click "Print Selected"
3. Configure layout (gear icon)
4. Open print dialog (Ctrl+P)
5. **Print directly from browser** OR **save as PDF and print from PDF viewer**
   - **Important**: When printing, use 100% scale (critical!) and highest quality/resolution
   - **Do not use auto-scaling** - codes will not work if scaled
   - When saving as PDF: ensure PDF viewer also prints at 100% scale

## OID Settings

**Resolution (DPI)**:
- 600 DPI (minimum)
- 1200 DPI (recommended)
- 2400 DPI (high-end printers)

**Pixels per dot**:
- 2 pixels (default)
- 3-4 pixels if codes not recognized

## Layout Presets

**List**: Full details, few albums per page

**Tiles**: Compact grid, many albums per page

## Troubleshooting

**Codes not recognized**:
1. Test with [OID table](https://github.com/thawn/ttmp32gme/blob/master/src/assets/images/oid-table.png)
2. If test fails: printer issue (try different printer/settings)
3. If test works: increase pixels per dot (3 or 4), use 1200 DPI
4. Must print at 100% scale, no auto-scaling

**Quality issues**:
- Clean print heads, replace ink/toner
- Use smooth white paper (avoid recycled)
- Enable graphics/image mode
- Try black-and-white vs color

## Printer Compatibility

Works best with:
- Laser printers (better than inkjet)
- 1200+ DPI resolution
- Windows: Use Microsoft Edge or save as PDF
- macOS: Chrome/Firefox work well
- Linux: Firefox recommended
